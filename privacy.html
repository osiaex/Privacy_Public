隐私权政策（Privacy Policy）

最后更新日期 / Last Updated：2025-11-14

1. 数据收集范围 / Scope of Data Collection

本扩展程序 AICHAT BUBBLE 可能会在运行过程中收集以下类型的数据：

  • 用户活动（User Activity）： 包括页面滚动、点击、鼠标位置或击键事件，用于改进气泡条动画与交互逻辑。
  • 网页内容（Website Content）： 如 ChatGPT 页面中的文字，仅用于生成本地气泡摘要。

2. 数据使用目的 / Purpose of Use

  • 改进扩展功能与交互体验。
    Improve the extension’s functionality and interactivity.
  • 优化 LLM 页面气泡条的视觉呈现。
    Enhance the visual experience of chat bubbles on LLM pages.

3. 数据存储与传输 / Data Storage and Transfer

当前版本的所有数据处理均在用户浏览器本地完成，扩展不会主动向开发者或第三方服务器发送任何信息。 **未来版本可能在用户明确同意后，将部分匿名化的用户活动数据上传至安全服务器，以用于统计分析和功能改进。**

All data processing in the current version occurs locally in the user's browser. **Future versions may, with explicit user consent, upload anonymized user activity data to a secure server for analytics and improvement purposes.**

4. 数据共享 / Data Sharing

扩展不会出售或向第三方共享个人可识别信息； 若未来进行数据上传，上传数据将严格匿名化，仅用于改进算法性能与用户体验。
The extension never sells or shares personally identifiable data; any future uploads will be strictly anonymized and used only for performance and UX improvement.

5. 用户控制 / User Control

用户可随时在浏览器扩展管理页面（chrome://extensions/）禁用或卸载本扩展，以停止所有数据收集与传输。 在未来版本中，如启用数据上传功能，将提供明确的开关选项与提示。
Users can disable or remove the extension at any time to stop all collection. Any future upload features will include an explicit opt-in switch and clear notice.
